The number of days for M-month faculty with Y annual salary to earn overload Z:
( Z $ ) ÷ ( Y $ year ) x ( M months year ) ÷ ( 12 months year ) x ( 52 weeks year ) x ( 40 hours week ) ÷ ( 8 hours day ) = ( 52 x 40 12 x 8 ) ( M x Z Y )
The number of days for M-month faculty with Y annual salary to earn overload Z: (Z $) ÷ (Y $/year) x (M months/year) ÷ (12 months/year) x (52 weeks/year) x (40 hours/week) ÷ (8 hours/day) = (52 x 40 ÷ 12 ÷ 8) x (M x Z ÷ Y).
