NSF Budget Planning

NSF Proposal Budget Planner

Plan one NSF proposal budget period in a worksheet-style format before entering values into the official proposal system. Personnel rows compute salary, fringe, direct cost, indirect cost, and total cost, while non-personnel budget categories remain editable and print-ready.

Budget Settings

Leave row fringe and indirect rate cells blank to use these defaults. Clear a row's IDC checkbox when indirect cost should not apply.
For categories with special indirect-cost treatment, enter only the allowable direct-cost base here or clear IDC for that row.
Reset clears the current worksheet. Copy Text Summary copies a plain-text budget summary to the clipboard. Print / Save as PDF opens the browser print dialog. Help opens a quick reference for the worksheet structure and calculations.

How This Planner Works

Purpose

This tool helps a PI plan one NSF proposal budget period in a worksheet-style format before entering values into the official NSF budget system.

Personnel Rows

  • Enter one row per person.
  • Requested salary is calculated as annual base × budgeted months ÷ 12.
  • Fringe and indirect costs are then added to produce the total row cost.

Rates

  • Default fringe and indirect rates are editable at the top of the page.
  • Leave row override cells blank to use the default rate.
  • Clear a row’s IDC checkbox when indirect cost should not apply.

Worksheet Structure

  • Senior personnel are kept separate from other personnel categories.
  • Non-personnel budget lines remain manual but can still compute indirect cost.
  • Participant support, equipment, and other categories remain distinct so IDC treatment can be controlled row by row.

Outputs

  • The top summary cards show total direct costs, total indirect costs, and total budget.
  • The budget totals section rolls the worksheet into an NSF-style budget summary.
  • Copy Text Summary creates a plain-text worksheet summary for email or planning notes.
  • Print / Save as PDF creates a print-friendly version of the worksheet.
