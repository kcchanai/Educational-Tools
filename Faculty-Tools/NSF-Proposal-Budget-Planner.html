NSF Budget Planning

NSF Proposal Budget Planner

Plan one NSF proposal budget period in a worksheet-style format before entering values into the official proposal system. Personnel rows capture person-months, salary-before-fringe, and fringe, with an optional Budget Helper for release time and overload estimates. The worksheet carries calculations to the nearest cent internally and displays rounded-up whole-dollar totals for direct costs, modified total direct costs, and indirect cost for the full budget period.

Budget Settings

Enter salary-before-fringe and fringe directly for each person. Clear a row's IDC checkbox when indirect cost should not apply to that direct-cost line.
For categories with special indirect-cost treatment, enter only the allowable direct-cost base here or clear IDC for that row. MTDC is calculated from the direct-cost lines that remain eligible for IDC.
Reset clears the current worksheet. Copy Text Summary copies a plain-text budget summary to the clipboard. Copy Shareable Link copies a URL with the current worksheet state. Print / Save as PDF opens the browser print dialog. Help opens a quick reference for the worksheet structure and calculations.

How This Planner Works

Purpose

This tool helps a PI plan one NSF proposal budget period in a worksheet-style format before entering values into the official NSF budget system.

Personnel Rows

  • Enter one row per person.
  • Enter CAL, ACAD, and SUMR person-months for both Sections A and B.
  • Enter the salary amount before fringe and the fringe amount separately for each person.
  • Use Budget Helper on a row to derive person-months, salary before fringe, and fringe from release time and overload inputs.
  • Budget Helper calculates ACAD months as release TE divided by 3, and SUMR months as summer instructional TE divided by 3 plus non-instructional overload days divided by 21.
  • Summer person-month calculations assume 63 work days across 3 months of summer.

Rates

  • The default indirect cost rate is editable at the top of the page.
  • Leave a row IDC override blank to use the default indirect rate.
  • Clear a row's IDC checkbox when indirect cost should not apply.

Worksheet Structure

  • Section A is senior personnel and Section B is one combined other-personnel list.
  • Sections D, E, F, and G share one table, with participant support split into F1-F4 and other direct costs split into G1-G6.
  • Each D, E, F, and G row includes Funds Requested, an IDC checkbox, and an optional IDC override rate.

Rounding and Totals

  • Money calculations are carried internally to the nearest cent.
  • Displayed dollar amounts are rounded up to the nearest whole dollar.
  • The Budget Totals section uses the rounded displayed line values, and indirect costs are calculated from the rounded-up MTDC value.

Outputs

  • The budget totals section rolls the worksheet into an NSF-style budget summary.
  • Copy Text Summary creates a plain-text worksheet summary for email or planning notes.
  • Copy Shareable Link copies a URL containing the current worksheet state.
  • Print / Save as PDF creates a print-friendly version of the worksheet.
