How This Planner Works

Purpose

This tool helps you plan one or more NSF proposal budget periods in a worksheet-style format before entering values into the NSF budget worksheet or Research.gov.

Personnel Rows

  • Section A uses five shared senior-person rows plus a sixth Others row with a quantity field.
  • The A1 through A5 senior-person names are shared across the whole proposal and can be edited from any editable year tab.
  • Row 7 in Section A totals rows 1 through 6 automatically.
  • Section B uses six fixed NSF categories with a quantity field and Funds Requested (Before Fringe).
  • Only B1 and B2 track CAL, ACAD, and SUMR person-months.
  • Use Budget Helper on the five named Section A rows to derive person-months, salary before fringe, and fringe from Desired Release Time, Desired Instructional Summer Overload, and Desired Non-Instructional Summer Overload Days.
  • Each Section A Budget Helper row can use Summer Instructional Rate Source (by rank) to choose a live faculty rate, a live lecturer rate, or a custom rate entered in the adjacent Cost per Instructional Overload TE/Credit cell.
  • Budget Helper calculates ACAD months as Desired Release Time (TE) or Buyout (Credit) divided by 3, and SUMR months as Desired Instructional Summer Overload (TE or Credit) divided by 3 plus Desired Non-Instructional Summer Overload Days divided by 21.
  • Release or buyout salary uses annual salary divided by Required Annual Credit Load. 9-Month (UHCCs) and 11-Month (UHCCs) lock that value to 27 or 36, while 9-Month (UH 4-Year) and 11-Month (UH 4-Year) leave it blank and editable.
  • For non-instructional summer overload days, 9-Month (UH 4-Year) uses the same calculation basis as 9-Month (UHCCs), and 11-Month (UH 4-Year) uses the same calculation basis as 11-Month (UHCCs).
  • Summer person-month calculations assume 63 work days across 3 months of summer.

Shared Settings and Rates

  • Organization, proposal number, PI / PD name, and Org. Rep. Name are shared across the whole proposal.
  • The Custom budget period label plus the on-campus and off-campus indirect cost rates belong to the active year tab and are edited in Active Budget Period Settings.
  • Money-entry fields show a non-editable dollar sign; type numbers only.
  • Use Reload Live ORS Defaults to refresh the planner's on-campus and off-campus indirect cost rates and the Budget Helper fringe defaults from the UH ORS rate page.
  • Live summer instructional pay rates require a configured proxy URL. When unavailable, Budget Helper still supports manual entry in the adjacent Cost per Instructional Overload TE/Credit cell.
  • Each Section A, B, D, E, F, and G budget row can use On-campus, Off-campus, or Exempt IDC treatment.
  • Personnel costs no longer default into the on-campus IDC base if you explicitly mark a Section A or B row as Off-campus or Exempt.

Non-Instructional Overload Formulas

The helper's non-instructional overload day calculations correspond to these formulas:

9-Month Faculty

($ overload) / ($ salary / year) x (9 months / year) / (12 months / year) x (52 weeks / year) x (40 hours / week) / (8 hours / day) = ((52 x 40) / (12 x 8)) x ((9 x overload) / salary)

11-Month Faculty

($ overload) / ($ salary / year) x (52 weeks / year) x (40 hours / week) / (8 hours / day) = ((52 x 40) / 8) x (overload / salary)

Multi-Year Workflow

  • A new proposal starts with three editable budget periods by default.
  • Use Add Blank Year to create another empty budget period.
  • Use Duplicate Year 1 to add a new budget period prefilled with Year 1 values.
  • Remove Active Year deletes the selected editable tab.
  • At least one budget period always remains.
  • Switching tabs preserves the active year's worksheet values, Budget Helper values, and totals.
  • Reset clears only the active year and keeps shared proposal info plus the other year tabs intact.
  • When two or more years exist, a read-only Cumulative tab appears and sums each row across the editable year tabs.
  • The sticky Budget Period Tabs card near Active Budget Period Settings lets you switch between year tabs without scrolling back to the top.

Worksheet Structure

  • Section A is a fixed six-row senior-personnel table. Section B is a fixed six-row NSF other-personnel table.
  • Sections D through M follow the NSF worksheet order, with separate sections for equipment, travel, participant support, other direct costs, indirect costs, fee, amount requested, and cost sharing.
  • Section D includes a multiline equipment description field so you can list multiple equipment items exceeding $10,000.
  • Each A, B, D, E, F, and G budget line includes an IDC Treatment selector.
  • Section F includes participant-support line items, a Total Number of Participants input, and a Total Participant Costs summary row.
  • Section I shows separate on-campus and off-campus MTDC bases, the rate used for each base, and total indirect costs.
  • Section M includes both Cost Sharing Proposed Level and Agreed Level if Different.

Rounding and Totals

  • Money calculations are carried internally to the nearest cent.
  • Displayed dollar amounts are rounded up to the nearest whole dollar.
  • The NSF total lines use the rounded displayed line values, and indirect costs are calculated from the rounded-up MTDC value.

Outputs

  • The active worksheet displays NSF-style totals inline as you work through Sections A through M.
  • When available, the Cumulative tab shows a summed multi-year view across all editable year tabs.
  • Copy Text Summary creates a plain-text multi-year proposal summary for email or planning notes.
  • Download Spreadsheet creates an XLSX workbook with proposal metadata, one worksheet per budget period, a summary sheet, and a cumulative sheet when available.
  • The spreadsheet includes native Excel formulas for worksheet totals and Budget Helper outputs so totals can recalculate inside Excel.
  • Exported spreadsheet cells use wrapped text formatting to make long labels and notes easier to read.
  • Copy Shareable Link copies a URL containing the current proposal state.
  • Print / Save as PDF creates a print-friendly multi-year proposal summary, including the cumulative view when available.
