Enter the values for the Delta resistors (Ra, Rb, Rc).

Wye Resistors

Formulas:

R1 = (Rb * Rc) / (Ra + Rb + Rc)

R2 = (Ra * Rc) / (Ra + Rb + Rc)

R3 = (Ra * Rb) / (Ra + Rb + Rc)

Wye (Y) to Delta (Δ)

Enter the values for the Wye resistors (R1, R2, R3).

Delta Resistors

Formulas:

Ra = (R1R2 + R2R3 + R3R1) / R1

Rb = (R1R2 + R2R3 + R3R1) / R2

Rc = (R1R2 + R2R3 + R3R1) / R3
